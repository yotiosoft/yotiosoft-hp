    Cities Box

    Windows
    Mac
    HSP3
    C++
    完全無料の都市開発ゲーム！
    まだまだ機能は少ないですが、道路の敷設、建物の建設、市民の要求に応えるなど、
    必要最低限の機能は揃っています。
    機能が揃ったHSP3版※と、現在開発中かつ新しい機能を体感できるC++版があります。
    ※HSP3版はWindowsのみ対応。

    非リアが手を握るのはだめですか

    公開日：2019年12月24日
    Windows
    HSP3
    非リア充を十字キーで操作して、画面左側から流れてくるリア充を蹴飛ばすゲーム。
    非リア充に近づくと、その非リア充と仲間になり手を繋ぐことができます。 岩に当たると仲間の非リア充が一人減ります。 最後に、非リア充が残り1人の状態で岩につまづくと、非リア充が0人になりゲームが終了します。 このゲームには制限時間はありません。
