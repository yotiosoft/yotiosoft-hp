Webツール集「Yapps」を公開しました
新作
 公開日：2021年12月1日

本日、Webブラウザ上で動作する便利ツール集サイト「Yapps」を新たに公開しました。
Yappsは、Web制作、レポート作成、スケジュール管理など様々な用途での利用を想定したWebツール集です。
Webブラウザ上で動作しますので、インストール不要で今すぐ利用可能です。

Yapps
https://yapps.yotiosoft.com/



利用可能なアプリ
現時点で以下の5つのアプリが使用可能です。
  • 全角半角判定
  • 全角半角変換
  • 英数字を反転
  • メモ帳
  • 文章読み上げ
なお、アプリは今後も追加していきます。現在、画像からRGB値を取得するアプリを試作中です。
利用条件
ご利用のブラウザが以下の条件を満たしている必要があります。
  • JavaScriptが有効化されていること。
  • Cookie類似技術の使用が許可されていること。
  • HTML5に対応していること。
動作環境
以下のブラウザで動作確認済みです。
  • Google Chrome (Windows / macOS / iOS / Android / Chrome OS)
  • Firefox (Windows)
  • Microsoft Edge (Windows)
  • Safari (macOS / iOS)
