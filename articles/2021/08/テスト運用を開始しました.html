    お知らせ
     公開日：2021年8月11日

    本日より、新ホームページにおけるテスト運用を開始しました。
    まだ作成途中ですので、いくつかアクセスできないページがありますのでご了承ください。
    また、当サイトは現時点ではまだスマホ用のページは用意しておりません。
    画面幅が狭い端末では小さく表示されてしまいます。近日中に対処します。

    コンセプト
    シンプルで、軽量で、情報にたどり着きやすいサイトをコンセプトに作成しました。
    今回はPageSpeed Insightsを参考にしながら作成しています。
    だいぶ軽量になりましたが、まだまだ改善の余地ありです。
    サーバについて
    Github Pagesを利用しています。
    リポジトリはこちら
    移転日について
    このサイトへのホームページの移転は10月1日を予定しています。
    それまではテスト運用をしつつ、随時ページの作成や調整を続けます。
    表示テストを兼ねて公開していますので、時折表示が崩れたりすることもあるかもしれませんがご容赦ください。
    今回のリニューアルでは、以前の構成よりも目的のページにたどり着きやすいこと、ページの読み込みが軽いこと、作品紹介とサポート情報を分かりやすく整理することを特に重視しています。
    そのため、テスト運用中は見た目の装飾よりも、まずはリンク構造や情報配置、ページ分割の方針が妥当かどうかを確認しながら進めていました。
    GitHub Pages を採用したのも、静的ページ中心で運用しやすく、更新履歴を管理しやすいこと、そして余計なサーバ管理の負担を増やさずにコンテンツ作成へ集中できることが理由です。
    一方で、実際に公開してみないと分からない問題も多く、ブラウザ差異、画像サイズ、メニューの追従挙動、スマートフォンでの見え方などは、運用しながら調整する前提でした。
    テスト運用という形で早めに公開したのは、完成を待ってから一気に切り替えるよりも、少しずつ確認しながら改善していくほうが結果的に安定した移行につながると考えたためです。
    その意味では、このお知らせ自体も移転作業の節目を記録するページとして残しています。
    完成版だけを見ると見えにくい試行錯誤の過程も、後からサイト全体を振り返る上では重要な記録になると考えています。
