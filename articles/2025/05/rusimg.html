画像処理ツール rusimg を公開しました
新作
 公開日：2025年5月6日

Rust で作られたコマンドライン向け画像処理ツール「rusimg」を公開しました。
BMP, JPEG, PNG, WebP 形式に対応し、それぞれのフォーマット間の画像変換や、リサイズ処理、トリミング処理、グレースケール化といった処理をコマンドライン上で一括に行えます。
https://crates.io/crates/rusimg



機能

※詳細は下記の特設サイトをご覧ください。
https://rusimg-app.yotiosoft.com/

  • 画像フォーマット変換 (-c)
  • リサイズ (-r)
  • トリミング (-t)
  • グレースケール化 (-g)
  • 画像プレビュー機能 (-v)

Rusimg にはコマンドライン上で画像をプレビューする機能があります。
この機能を使用した場合、画像を操作する際に、変換後の画像を確認することができます。

サンプル

ディレクトリ内の png 画像に対して webp 形式に変換し、50% にリサイズする 
$ rusimg -i ./image/*.png -o ./image/output -c webp -r 50


インストール方法

cargo を使用してインストールできます。事前に Rust のインストールが必要です。
$ cargo install rusimg


ライブラリクレート

rusimg は画像処理ライブラリとして librusimg を公開しています。
こちらも併せてご利用ください。
https://crates.io/crates/librusimg
