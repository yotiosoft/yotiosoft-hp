    • dptran v2.1.1をリリースしました
    アップデート
     公開日：2024年12月24日

    本日、コマンドライン上でDeepL翻訳を実行するCLIツール「dptran」のアップデート版であるv2.1.1を公開しました。



    dptranはコマンド引数での原文の指定のほか、対話形式での翻訳やパイプラインからの原文受け取りも可能です。

    翻訳にはDeepL APIを用います。ご利用の場合は、事前にDeepL APIにてAPIキーの取得をお願いいたします。
    DeepL APIはフリープランの場合は無料で取得可能ですが、月50万文字までという制限がございますのでご注意ください。
    更新内容
    • DeepL APIの言語一覧リストに載っていない言語への対応。
      • DeepL APIでは、テキスト翻訳のみ対応してドキュメント翻訳が非対応の言語は言語一覧リストに含めない仕様であるため、従来の dptran ではそれらの言語に対応できませんでした。
      • 今回のアップデートでは、言語一覧リストに載っていない言語をライブラリ側で対応させています。
      • これによりアラビア語、繁体字中国語への翻訳が可能になりました。
    インストール方法
    バイナリクレート（ツール本体）のインストール：
    $ cargo install dptran

    プロジェクトへのライブラリクレートの追加 (Cargo.toml)： 
    [dependencies]
dptran = { version = "2.1.1", default-features = false }
    現状は crates.io からのインストールのみとなります。
    ご利用には rustup または cargo のインストールが必要です。こちらをご覧ください。
    https://crates.io/crates/dptran
